# Invitación Interactiva de XV Años - Paulina 🌸

Este proyecto es una invitación digital personalizada e interactiva diseñada para un evento de XV años. El enfoque principal fue crear una experiencia de usuario fluida, elegante y funcional, optimizada para dispositivos móviles.

## 🚀 Características Principales

- **Diseño Responsive:** Adaptado para visualizarse correctamente en cualquier dispositivo móvil (Mobile-First).
- **Cuenta Regresiva:** Reloj dinámico en tiempo real que indica los días, horas, minutos y segundos restantes para el evento.
- **Efectos de Scroll (Fade-in):** Implementación de animaciones suaves mediante el API de JavaScript `Intersection Observer` para mejorar la estética visual al navegar.
- **Integración con Google Maps:** Botón con enlace directo a la ubicación exacta del evento.
- **Confirmación por WhatsApp:** Enlace directo con mensaje predefinido para facilitar la confirmación de asistencia.
- **Sección de Regalos Interactiva:** Bloque dinámico que permite mostrar u ocultar los datos bancarios (Alias/CBU) para mantener la limpieza visual del diseño.
- **Gráficos Animados:** Uso de SVG animado con CSS para elementos decorativos (diamante).

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica por bloques.
- **CSS3:** Diseño personalizado, Flexbox, animaciones con `@keyframes` y manejo de variables (`:root`).
- **JavaScript (Vanilla JS):** - Lógica de la cuenta regresiva con `setInterval`.
    - Manipulación del DOM para la interactividad de la sección de regalos.
    - `Intersection Observer API` para las animaciones de aparición al hacer scroll.
- **Google Fonts:** Uso de *Montserrat* y *Playfair Display* para un contraste tipográfico sofisticado.
